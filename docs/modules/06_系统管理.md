# 系统管理

> 本文件是 `系统管理` 模块的独立规格。Codex 开发本模块时，应同时读取 `AGENTS.md`、`docs/00_项目总览.md`、通用架构规范和对应阶段计划。

## 14. 模块六：系统管理

### 14.1 模块定位

系统管理负责配置、环境检测、日志、运行监控、备份恢复、操作记录。

它是保证系统能稳定运行和可排查问题的基础模块。

### 14.2 页面结构

```text
系统管理
├── 基础设置
├── 环境检测
├── 交易设置
├── 策略设置
├── 日志中心
├── 运行监控
├── 备份恢复
└── 操作记录
```

### 14.3 基础设置

#### 前端功能

字段：

| 字段 | 说明 |
|---|---|
| QMT 路径 | 本地 QMT / MiniQMT 路径 |
| 账户 ID | QMT 账户 ID |
| 数据库路径 | SQLite 文件路径 |
| 策略目录 | strategies/user/ |
| 备份目录 | backups/ |
| 启动自动连接 | 是/否 |
| 启动自动同步 | 是/否 |

操作：

```text
保存设置
测试路径
恢复默认
```

#### 后端功能

服务：`config_service.py`

职责：

1. 读取配置。
2. 保存配置。
3. 校验路径。
4. 不在前端暴露敏感信息。

#### API

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/system/config` | 获取配置 |
| PUT | `/api/system/config` | 保存配置 |
| POST | `/api/system/config/test-path` | 测试路径 |

### 14.4 环境检测

#### 前端功能

检测项：

```text
QMT路径是否存在
MiniQMT是否启动
是否登录账户
xtquant是否可导入
Python版本是否兼容
是否能查询资产
是否能查询持仓
是否能获取行情
交易接口是否被保护
```

显示：

```text
检测项、结果、说明、修复建议、技术详情
```

#### 后端功能

服务：`environment_check_service.py`

职责：

1. 检测 Python 环境。
2. 检测 xtquant。
3. 检测 QMT 路径。
4. 检测连接状态。
5. 检测查询资产/持仓/行情能力。
6. 检测交易接口保护状态，确认真实 QMT 只读阶段不会真实下单。

#### API

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | `/api/system/env/check` | 创建环境检测任务 |
| GET | `/api/system/env/results` | 检测结果 |

### 14.5 交易设置

字段：

```text
默认单笔金额
最大单笔金额
是否下单前确认
默认委托方式
委托价格偏移
是否启用测试隔离模式
```

后端保存到 `app_config`。

### 14.6 策略设置

字段：

```text
策略默认超时
策略运行间隔
是否盘中自动运行
默认日志级别
策略最大日志大小
```

第一版自动运行可以先做开关和配置，不一定立即实现复杂调度。

### 14.7 日志中心

日志类型：

```text
同步日志
策略日志
回测日志
交易日志
系统日志
```

功能：

1. 分页查询。
2. 按模块筛选。
3. 按等级筛选。
4. 按日期筛选。
5. 复制日志。
6. 导出日志。
7. 清理过期日志。

### 14.8 运行监控

展示：

```text
运行中任务
最近失败任务
数据库大小
日志大小
策略目录大小
慢接口
慢任务
最近错误
```

作用：

1. 发现系统卡顿原因。
2. 发现数据库过大。
3. 发现日志过多。
4. 发现同步或回测异常。

### 14.9 备份恢复

备份内容：

```text
SQLite 数据库
strategies/user/ 策略目录
配置文件
重要日志
回测结果
```

操作：

```text
立即备份
恢复备份
打开目录
删除备份
```

约束：

1. 恢复备份前必须确认。
2. 恢复前先自动创建当前快照。
3. 备份任务必须有进度。
4. 策略目录必须备份。

### 14.10 操作记录

记录操作：

```text
修改配置
环境检测
连接QMT
同步数据
新建策略
保存策略
运行策略
启动回测
手动下单
信号下单
撤单
备份恢复
```

字段：

```text
操作时间、模块、操作类型、操作对象、结果、中文说明、技术详情
```

### 14.11 系统管理验收标准

1. 能保存基础配置。
2. 能做环境检测并给中文建议。
3. 能设置交易参数。
4. 能设置策略参数。
5. 能查看和导出日志。
6. 能查看运行监控。
7. 能备份和恢复数据库、策略、配置。
8. 所有关键操作都有记录。
